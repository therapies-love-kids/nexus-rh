# Commit Semântico

## Formato

```<tipo>: <assunto>```

## Tipos

* `ADD` - Adição
* `MOD` - Modificação
* `DEL` - Remoção
* `FIX` - Correção

## Exemplos

### ADD

* Criação de uma tela de atividades - `ADD: Tela de atividades`

### MOD

* Alteração dos dados exigidos durante o processo de registro de um novo usuário - `MOD: Dados exigidos na tela de registrar usuário`

### DEL

* Remoção de um componente antigo de tabela, já não mais utilizado - `DEL: Componente antigo de tabela`

### FIX

* Correção do redirecionamento no botão de voltar na tela de profissionais - `FIX: Redirecionamento na tela de profissionais`
